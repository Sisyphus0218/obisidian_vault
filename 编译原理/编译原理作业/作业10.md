练习6.1.4：考虑龙书图 6-22 的翻译方案，翻译赋值语句  
$x = a[b[i][j]][c[k]];$  
并给出注释语法分析树。

答：

```
t1 = i * bi.width
t2 = j * bj.width
t3 = t1 + t2
t4 = b[t3]
t5 = t4 * ab.width
t6 = k * ck.width
t7 = c[t6]
t8 = t7 * ac.width
t9 = t5 + t8
t10 = a[t9]
x = t10
```
